# Password Security Validator

This project is a Python-based password protection system designed to improve application security by validating passwords and simulating common attack scenarios.

## 🔐 Features
- Password strength validation (length, uppercase, numbers, special characters)
- Brute force attack simulation with login attempt limits
- SQL injection input detection
- User-friendly feedback messages

## 🧪 Testing
The system uses both:
- Black-box testing (input/output validation)
- White-box testing (logic verification)

## 🚀 Purpose
This project demonstrates secure software development practices and aligns with Software Quality Assurance (SQA) principles by ensuring reliability, validation, and protection against common vulnerabilities.

## 🛠️ Technologies Used
- Python
- Regular Expressions (`re`)
- Basic security validation techniques

## 📌 Author
Developed as part of a software engineering assignment focused on security and testing.
